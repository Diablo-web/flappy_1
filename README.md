## How to install and run locally ?
Clone the project
```
$ git clone https://github.com/Diablo-web/flappy.git
```
Go to the project directory
```
$ cd flappy
```
Install dependencies
```
$ npm install
```
To run this project
```
$ npm run start
```
**Note: this is a one-way operation. Once you `eject`, you can't go back!**
